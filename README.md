## Тема: Web-приложение для ведения и оценки групповых работ
## Цель работы: Разработка web-приложения для взаимодействия преподавателя и студентов при ведении групповых работ

## Как запустить?
```
docker compose up -d
```
## О приложении

### Stack
#### Frontend: 
+ React
+ MUI
+ Typescript
+ Node.js
+ Docker

#### Backend:
+ Node.js
+ Typescript
+ Express.js
+ Docker
